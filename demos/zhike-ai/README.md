# 电商智能客服系统 · Web 端原型

> **产品定位**：面向中小电商商家的轻量级 AI 智能客服系统
> **使用场景**：商家店长 / 客服主管在 PC 端管理工作台
> **设计风格**：活泼年轻橙（与移动端保持一致）

---

## 一、原型概览

本目录是 5 个 Web 端页面的 HTML 原型，模拟客服主管在 PC 端管理整个客服团队和数据。

**设计语言**：Vue 3 + Element Plus 风格的视觉规范
- 左侧固定导航栏（220px）+ 右侧主区域
- 顶部 Header（60px）+ 面包屑导航
- 信息密度高，表格 + 图表 + 卡片化布局
- 主色 `#FF6B35` 活力橙 + 辅色 `#FFB627` 阳光黄

**技术栈**：HTML + 原生 CSS + ECharts（图表）+ Alpine.js（弹窗交互）
- 零编译，浏览器双击 `login.html` 即可预览
- 推荐 1440px 以上分辨率查看

---

## 二、页面清单

| # | 文件 | 页面 | 关键功能 |
|---|------|------|----------|
| 1 | `login.html` | 登录页 | 左右分栏 + 品牌区 + 三种登录方式（账号/手机/扫码） |
| 2 | `dashboard.html` | 工作台 Dashboard | 8 个核心指标 + 实时活动流 + 多图表 + 系统状态 |

| 5 | `analytics.html` | 数据分析 BI | 8 指标 + 8 图表 + AI 智能洞察 + 报表中心 |

```
                  #9CA3AF   弱化文字
边框（Border）     #E5E7EB   主边框
                  #F3F4F6   浅边框
```

### 3.2 布局结构

```
┌─────────────────────────────────────────────────┐
│ Sidebar  │  Top Header  (面包屑 · 搜索 · 工具)   │
│  220px   ├─────────────────────────────────────┤
```

### 3.3 通用组件

- **圆角**：卡片 12px、按钮 8px、输入框 8px、标签 4-6px
- **阴影**：极简风格（`0 1px 3px rgba(0,0,0,0.06)`）
- **字体**：14px 为主、12px 辅助、22-28px 标题
- **按钮高度**：常规 36px、小 28px、大 44px
- **表格**：行高 14px*1.5，悬停浅灰，激活行橙色背景

### 3.4 状态色

- 成功 `#10B981`（绿色）— 已完成、已解决
- 警告 `#F59E0B`（黄色）— 待处理、重要
- 危险 `#EF4444`（红色）— 紧急、超时
- 信息 `#3B82F6`（蓝色）— 进行中

## 四、页面详细说明

### 4.1 登录页（login.html）

**设计要点**
- 左右分栏布局（920×560）
- 左侧：**品牌区**（橙色渐变 + 装饰光斑 + 特性标签）
- 右侧：**表单区**（白色 + 3 种登录方式 Tab + 表单 + 第三方登录）

**包含元素**
- 品牌 Logo + 名称
- 4 个特性标签（1秒响应 / 87% 解决率 / 24h 在线 / 多平台）
- 表单 Tab：账号密码 / 手机验证 / 扫码登录
- 验证码 + 7天免登录 + 忘记密码
- 4 个第三方登录图标
- 立即注册 / 企业试用

**响应式**
- < 768px 自动隐藏左侧品牌区

---

### 4.2 Dashboard 工作台（dashboard.html）

**设计要点**
- 这是商家每天打开的第一个页面，信息密度最高
- 顶部 8 个**核心指标卡**（2 行 × 4 列），每个卡都有图标 + 趋势
- 中部三列布局：左侧大趋势图 + 右侧实时活动流
- 下部三列布局：饼图 + 客服工作量 + 待办事项
- 底部双列：热点问题 TOP10 + 系统状态

**8 个核心指标**
| 指标 | 数值 | 趋势 |
|------|------|------|
| 今日对话量 | 1,286 | ↑ 12.5% |
| AI 解决率 | 87.3% | ↑ 3.2% |
| 客户满意度 | 4.8/5 | ↑ 0.2 |
| 平均响应时长 | 1.2s | ↓ 0.3s |
| 人工介入率 | 12.7% | ↓ 2.1% |
| 转接成功率 | 94.5% | ↑ 1.8% |
| 促成交易额 | ¥38.6k | ↑ 22.1% |
| 知识库命中率 | 87% | ↑ 5.0% |

**实时活动流**
- 5 分钟内的动态（新会话、AI 解决、转人工、工单关闭、FAQ 新增、评价）

**系统状态卡**
- 5 个依赖服务的健康状态（Qwen / Qdrant / PostgreSQL / OSS / 第三方渠道）

---

### 4.3 会话管理（conversations.html）

**设计要点**
- 表格化设计（替代移动端的列表+详情），适合批量操作
- **多维筛选**：渠道 + 状态 + 处理人 + 时间
- 点击任意行 → 右侧**抽屉滑出**显示完整对话
- 抽屉宽 540px，含客户信息卡 + 消息流 + AI 推荐 + 快捷回复 + 输入框

**表格列**
1. 选择框
2. 客户（头像+姓名+VIP等级+订单数）
3. 最近消息预览
4. 渠道（带 emoji 标签）
5. 处理人
6. 状态（带圆点）
7. 等待时长
8. 开始时间
9. 操作（查看/转接/结束）

**抽屉详情**
- 客户完整画像（6 项指标）
- 标签、备注、订单详情快捷入口
- AI 推荐答案（含置信度、来源）
- 消息气泡（客户左、客服右、AI 黄）

---

### 4.4 知识库管理（knowledge.html）

**设计要点**
- 左侧**分类树**（可展开/收起）+ 右侧 FAQ 表格
- 表格含**完整元数据**：问题+答案预览、分类、扩展问法、命中数、命中率、更新时间
- ⚠️ **待优化标记**：命中率 < 50% 的条目用黄色高亮 + 优化按钮
- 点击"新增/编辑" → 居中**模态弹窗**（720px 宽）

**编辑器特色**
- 类 Word 工具栏（粗体/斜体/列表/链接/图片）
- 实时字数统计 + 关键词自动识别
- 支持 Markdown + 变量插值（`{商品名称}` `{订单号}`）
- 自动保存提示
- 扩展问法 chips 输入（回车添加）
- 关联商品选择

**分类树结构**
```
📋 全部问题 (286)
├── 🛒 售前咨询 (86)
│   ├── 商品咨询 (28)
│   ├── 价格咨询 (22)
│   ├── 库存咨询 (18)
│   └── 活动咨询 (18)
├── 🔧 售后服务 (124)
│   ├── 退换货 (56)
│   ├── 退款 (42)
│   └── 投诉建议 (26)
├── 🚚 物流配送 (48)
├── 💳 支付问题 (18)
└── 💬 其他 (10)
```

## 九、后续迭代方向

1. **深色模式** —— Web 端更适合做暗色主题
2. **大屏可视化** —— 单独做 1920×1080 的大屏看板

4. **权限细分** —— 客服只能看自己，主管看团队，管理员看全部
5. **快捷键** —— Ctrl+K 搜索、Esc 关闭弹窗等
6. **拖拽自定义看板** —— 允许用户自定义 Dashboard
7. **移动端适配** —— Web 端在 Pad 上的横屏体验
